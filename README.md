# 🐍 Snake Game in Java

This is a classic **Snake Game** developed in Java using **Swing** and **AWT** libraries. 

## 🎮 Features

- Classic snake gameplay
- Score tracking
- Game Over screen
- Keyboard controls using arrow keys
- GUI built with Java Swing and AWT
